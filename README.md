# Chat Color Manager

A RuneLite plugin that allows you to detect and recolor chat messages for better visibility and customization.

## Features

- **Chat Message Detection**: Automatically detects incoming chat messages
- **Color Recoloring**: Recolor messages based on configurable rules
- **Multiple Chat Types**: Handle different chat message types (public, private, game messages, etc.)
- **Customizable Rules**: Configure color rules per chat type
- **Real-time Updates**: Changes apply immediately to new messages

## Building

```
./gradlew build
```

## Installation

1. Build the plugin using the command above
2. Copy the `.jar` file from `build/libs/` to your RuneLite plugins folder
3. Restart RuneLite
4. Enable the plugin in the Plugin Hub

## Configuration

Configure color rules through the RuneLite plugin settings panel.
